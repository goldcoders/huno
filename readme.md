# Hugo with UnoCSS

## Requirements

- Hugo >= v0.70.0
- NodeJS >= ^14.18.2

## Installation

1. Clone Repo
```
git clone https://github.com/goldcoders/huno
```
2. Change Directory

```
cd huno
```
3. Install Dependencies

```
yarn
```

## Development

Run
```
yarn dev
```

## Production

Run
```
yarn build
```
